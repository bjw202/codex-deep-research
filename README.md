# codex-deep-research

Deep Research skill and example research outputs for Codex.

## Contents

- `.agents/skills/deep-research/`: the Deep Research skill instructions and role-specific references.
- `.codex/`: local agent configuration for researcher, journal, critic, and verifier roles.
- `docs/research/`: generated research artifacts and final synthesis reports.

## Purpose

This project stores a parallel deep-research workflow that separates primary research, journal summarization, critique, verification, and final synthesis. The final synthesis guidance emphasizes clear language, coherent decision logic, and reader-facing recommendations.
